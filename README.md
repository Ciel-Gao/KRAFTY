# KRAFTY
This repository contains the simulation and real data analysis code used in the paper: KRAFTY: Khatri-Rao Framework for Joint Cluster Recovery. 

## Overview
This repository provides:
1. Simulation experiments evaluating KRAFTY
2. Real data analysis code
3. Scripts for reproducing figures in the paper

## Files
1. Simulation/
  - The R scripts in this folder gives the simulation experiments. Files starting with `SimulationPlot_` produce plots for the simulation results. The `.RData` files contain saved simulation data.

2. Real data analysis/
  - This folder contains an R script for the real data analysis.
